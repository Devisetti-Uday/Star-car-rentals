Description:

Star Rentals is a simple PHP-based car rental web application.
Users can check available cars for selected dates, view details (model, seats, price), and book cars.

Features:

Dynamic car availability based on booking dates
MySQL database connection for managing cars and bookings
Styled using HTML, CSS (custom responsive design)
Date validation using HTML form required
Simple PHP backend logic
Local email sending support via mail()

Tech Stack:

Frontend: HTML, CSS
Backend: PHP
Database: MySQL (via XAMPP)

How to Run Locally:

Install XAMPP and start Apache & MySQL.
Place this project folder inside htdocs (e.g., C:\xampp\htdocs\happy).
Import the database file (rental.sql) into phpMyAdmin.
